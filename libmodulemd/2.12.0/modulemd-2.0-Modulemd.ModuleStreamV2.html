modulemd_module_stream_v2_search_profiles ()

GPtrArray *
modulemd_module_stream_v2_search_profiles
                               (ModulemdModuleStreamV2 *self,
                                const gchar *profile_pattern);

Parameters

self

This ModulemdModuleStreamV2 object.

  

profile_pattern

A globbing pattern to locate one or more profiles in this ModulemdModuleStreamV2 object. The names will be compared using fnmatch(3).

[nullable]

Returns

The list of ModulemdProfile objects whose name matched profile_pattern . This function cannot fail, but it may return a zero-length list if no matches were found. The returned profiles will be sorted alphabetically by profile name.
